## 📋 Описание

**Factory Tycoon Bot** — это интерактивная текстовая игра в Telegram, где вы выступаете в роли владельца фабрики. Ваша цель — развивать производство, зарабатывать деньги и достичь максимального рейтинга за 100 игровых месяцев.

### 🎮 Геймплей

- 🏭 **Управление фабрикой**: найм рабочих, улучшение экологии и качества техники
- ⚡ **Производственный цикл**: автоматический расчёт производительности
- 💰 **Экономика**: динамическая цена продажи, расходы, прибыль
- 🎲 **Случайные события**: более 10 уникальных событий с разными эффектами
- 📈 **Система репутации**: влияет на продажи и итоговый рейтинг
- 🏆 **Финал игры**: подсчёт очков и присвоение звания

### 🎯 Цель игры

Прожить 100 месяцев, развивая фабрику, и получить максимальный итоговый рейтинг:

| Рейтинг | Звание |
|---------|--------|
| 1000+ | 🏆 Легендарный предприниматель |
| 700-999 | ⭐ Мастер индустрии |
| 400-699 | 📈 Успешный бизнесмен |
| 200-399 | 📊 Начинающий фабрикант |
| 0-199 | 😰 На грани банкротства |

## ✨ Особенности

- ✅ **Современный aiogram 3** с асинхронной архитектурой
- ✅ **Полная игровая логика** с балансом и формулами
- ✅ **Интерактивное меню** с Inline-кнопками и FSM
- ✅ **Сохранение прогресса** в SQLite базе данных
- ✅ **Система улучшений** экологии, качества и оборудования
- ✅ **Чистый код** с разделением на модули (хэндлеры, фильтры, роутеры)
- ✅ **Легкое развертывание** на любом хостинге

## 🛠️ Технологии

- **[Python 3.9+](https://www.python.org/)** — язык программирования
- **[Aiogram 3](https://docs.aiogram.dev/)** — современная библиотека для Telegram API
- **[SQLite](https://www.sqlite.org/)** — встроенная база данных
- **[aiosqlite](https://github.com/omnilib/aiosqlite)** — асинхронный драйвер SQLite
- **[python-dotenv](https://github.com/theskumar/python-dotenv)** — управление переменными окружения
